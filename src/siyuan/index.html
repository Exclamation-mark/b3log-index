Block 编辑模式

以块来组织内容，每个块都可以缩放聚焦，支持横向排版和大纲式笔记书写

所见即所得

支持所见即所得编辑，在使用 Markdown 排版的同时也能让使用者将注意力集中在内容上

块级引用

支持 ((id)) 来引用其他 Markdown 文档中的块级元素（包括标题、段落、代码块、列表、块引用等），将卡片式双向链接和 Markdown 进行了语义层面的深度融合

本地离线

可以在离线断网的情况下使用，笔记数据完全保存在用户设备上
用户可使用第三方网盘同步，也可以使用思源自带的备份服务
