Block-level Reference

Supports ((id)) to reference block-level elements in other Markdown documents (including headings, paragraphs, code blocks, lists, block quotes, etc.), the card-style two-way link and Markdown are deeply integrated at the semantic level

Standard Markdown

Fully implement the GFM (GitHub Flavored Markdown) specification, which means that SiYuan has good compatibility, allowing the edited Markdown document to be universally used to the greatest extent

Rich block element functions

Various operations can be performed on block elements to meet different usage scenarios. Not only can you quickly share, use multiple methods to fold/expand, but also have bookmarks, naming, aliases, notes, and background color functions.

Instant Rendering

Supports an editing mode similar to Typora, only expands Markdown markers on the element where the cursor is located, which not only facilitates Markdown layout, but also allows users to focus on the content

Available offline

It can be used when offline, and the note data is completely saved on the user's device
Supports the WebDAV protocol, and the usage is almost the same as opening the local directory, providing users with more options for synchronizing data
